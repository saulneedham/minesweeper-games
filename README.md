# Minesweeper-Games
A collection of three Minesweeper implementations developed in Haskell, C, and Python to demonstrate three distinct programming paradigms. Each version includes custom gameplay mechanics, such as a "place bomb" feature to reveal or remove squares and a "remove row" ability to clear entire rows from the board.
